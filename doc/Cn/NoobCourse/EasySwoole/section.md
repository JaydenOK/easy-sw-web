## 组件设计
EasySwoole3.x版本采用组件分离设计,可自行选择需要的组件进行添加使用,框架本身只实现了基础的封装,框架变的更加简洁,轻便

### 组件列表
目前EasySwoole3.x开源组件如下:
* validate 验证组件
* http   http组件,实现了对http控制器的调度封装等
* mysqli 协程版mysqli组件
* rpc    rpc组件,可实现rpc分布式
* socket  socket组件,实现了对tcp/udp的控制器调度
* trace   trace组件,代码追踪
* component  基础组件库
* utility  工具类库
* verifyCode  验证码生成组件
* spl   spl组件
* curl  curl组件
