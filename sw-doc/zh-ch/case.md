# 用户与案例

如果您在使用Swoole，欢迎您将信息提交至此列表，让更多的用户了解Swoole的实际使用场景，以建立更好的Swoole生态。

## 期待您的信息

在[swoole-inc/report](https://github.com/swoole-inc/report/issues)提交一个`issue`，包括以下信息：

* 您所在公司、学校或组织
* 您所在的城市、国家
* 您的联系方式：微博、邮箱、微信
* 您将Swoole用于哪些业务场景

!> 如果联系方式不方便公开、修改相关信息或不想出现在此列表，请单独发送至邮箱：team@swoole.com
